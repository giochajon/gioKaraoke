How it works

Each MP3 is processed on the server using three open-source tools:

  • audio-separator — removes the lead vocalist using the UVR MDX-Net KARA model
  • Whisper — transcribes word-level timestamps from the original track
  • FFmpeg — merges the instrumental audio, background art, and ASS karaoke subtitles into an MP4

⚠ First run downloads AI models (~500 MB). Subsequent runs are faster. Files are not stored on the server — they are streamed directly to your browser.
