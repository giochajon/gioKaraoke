Mounting Songs

Set the SONGS_PATH variable in .env to your karaoke folder:

SONGS_PATH=/home/user/karaoke # Then start with: docker compose up -d

Or to generate a file list manually on the host:

find "$PWD" -type f \( -name "*.mp3" -o -name "*.mp4" \ -o -name "*.cdg" -o -name "*.zip" \) -print0 \ | xargs -0 realpath > filelist.txt
